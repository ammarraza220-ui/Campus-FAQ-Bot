# University Admission FAQ Chatbot

This project is a Python-based chatbot developed to assist students with university admission-related queries. The chatbot provides quick and automated responses to frequently asked questions such as admission requirements, application deadlines, eligibility criteria, fee structure, and available programs.

## Features
- Automated FAQ responses
- User-friendly interaction
- Fast query handling
- Python-based implementation
- Easy to customize and expand

## Technologies Used
- Python
- NLP techniques
- Chatbot logic and automation
